
=== Publishing own Package ===

1. Create an account on npmjs.com
dont use 'npm adduser' better go to website and register there

2. After you registered and confirmed your email
use 'npm login' to login as youself

3. Publish
use 'npm publish'
If an error happens try to change your 
package name inside package.json

4. After published you can install your
package as any other package via 'npm i ... '


=== Updating published package ===

1. Make a changes
2. Use 'npm version <major/minor/patch>'
e.g. 'npm version minor'
3. Publish
